# FriendsFanPageWebsite


<h2>Description</h2>
The Friends Fan Page is a responsive website developed using HTML5 and CSS3, aimed at teaching beginners the basics of these languages. It features a homepage, characters page, recognitions page, and gallery page. The website uses a grid system for layout and includes click functions for interactivity. Various CSS3 features, such as transitions and hover effects, are also showcased to enhance the design and user experience. Overall, it's an excellent practical guide for creating a simple yet effective website.
